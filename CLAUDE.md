# Campus AI Pitch

## Overview
Static HTML pitch page for university AI suite business — targeting campuses in Surabaya, Indonesia. This is a sales/marketing deliverable, not a software product.

## Purpose
Single-page mobile-first pitch site to share with university contacts via WhatsApp link. Demonstrates the value proposition of an AI-powered WhatsApp campus chatbot.

## Tech Stack
| Component | Technology |
|-----------|------------|
| Frontend | Static HTML + CSS + vanilla JS |
| Hosting | GitHub Pages |
| Language | Bahasa Indonesia |

## Architecture
Single `index.html` file with embedded CSS and JS. No build step, no dependencies.

## Key Patterns
- Mobile-first responsive design
- CSS scroll animations (no JS frameworks)
- WhatsApp chat mockup as hero visual
- All content in Bahasa Indonesia

## Commands
```bash
# Local preview
python -m http.server 8080
# Then open http://localhost:8080
```
