## What Is This?

A single-page static HTML site designed to pitch an AI-powered WhatsApp campus chatbot to university decision-makers. Shared via WhatsApp link, viewed on mobile.

## Target Audience

- Rektorat / IT division heads
- Kaprodi (program coordinators)
- BAA (academic admin bureau) staff

At: ITATS Surabaya, Unair, Ubaya, Petra
